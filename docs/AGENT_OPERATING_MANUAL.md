# Agent Operating Manual

## Grundsatz

Claude Code arbeitet phasenweise.

## Jede Phase braucht

- Ziel
- erlaubte Dateien
- verbotene Arbeiten
- Tests
- manuelle Prüfung
- Definition of Done

## Wenn unklar

Nicht raten. Erst kleine Annahme formulieren und Plan begrenzen.

## Wenn Nutzer "mach" sagt

Trotzdem nicht das ganze Projekt bauen. Nur die nächste kleinste sinnvolle Phase.

## Claude-Code-Best-Practice

Erkunden, Planen und Implementieren werden getrennt. Für unklare oder mehrdateiige Aufgaben wird Plan Mode bevorzugt.
