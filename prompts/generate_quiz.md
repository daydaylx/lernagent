# generate_quiz – Referenz-Template

Dieses Dokument beschreibt das Quiz-Format, das `app/explainer.py` im Modus `quiz` erzeugt.

## Ausgabeformat

```
1. [Frage]
   Antwort: ...

2. [Frage]
   Antwort: ...
```

## Vorgaben

- Mindestens 5 Fragen, maximal 10
- Einfache deutsche Sprache
- Keine erfundenen Informationen
- Unsicherheit offen markieren
- Fragen sollen Verständnis prüfen, nicht Auswendiglernen fördern

## Typische Fragekategorien

- Was macht diese Datei?
- Wann wird diese Funktion aufgerufen?
- Warum wird dieses Muster verwendet?
- Was könnte schiefgehen?
- Wie hängt diese Datei mit anderen zusammen?
